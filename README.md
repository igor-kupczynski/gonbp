# Go NBP

This is a go client for the NBP currency exchange rates API.

[Narodowy Bank Polski](http://www.nbp.pl/), or NBP, is the central bank of the Republic of Poland. Among other tasks it publishes the official exchange rate of Złoty (the Polish currency) against other major currencies. NBP provides an API to access the exchange rates. It is documented http://api.nbp.pl/en.html
